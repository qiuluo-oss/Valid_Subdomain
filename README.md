# 安装
pip3 install -r requirements.txt

# 使用

1、将需要验证的子域名粘贴到url.txt中

2、执行`python3 verify.py`即可

3、经华为云服务器测试，2500左右的域名需要2分钟就可以跑完，6100左右的域名需要。
